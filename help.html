Help Page

Smoke Installer is still in the "beta" version and likely won't have any in app problems. If you have a problem with installing Smoke Installer which is likely the only problem you'll have follow these instructions:

1:

Reread the requirements then install or update them

2:

Reinstall the latest version of Smoke Install App, delete anything from Smoke Installer outside of the app directory (any binaries or build/install scripts), then install again

3:

At this point you've tried everything, now you should make an issue on the Smoke Installer source code with the label "install problem", explain what is happening (errors, warnings) what OS and architecture you're using.
