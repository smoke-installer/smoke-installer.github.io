Install

Install on Linux

git install https://github.com/smoke-installer/smoke-installer-app.git

Linux

cd smoke-installer-app/

sudo python3 linux.py

Now you can open Smoke Installer by running:

Smoke_Installer

If you want to update Smoke Installer, run this command (it doesn't matter what OS you have)

smoke-install-update

Windows & MacOS

Windows 10 and MacOS support is being worked on and might be available by the start of summer 2021
